## 문법

`@copyright <저작권 텍스트>`

## 개요

@copyright 태그는 파일 개요 주석에서 저작권 정보를 문서화하는 데 사용됩니다. 이 태그는 [@file 태그](./file.md)와 함께 사용하십시오.

## 예시

```js
/**
 * @file 이것은 나의 멋진 스크립트입니다.
 * @copyright Michael Mathews 2011
 */
```
