## 문법

`@summary 요약이 여기 들어갑니다.`

## 개요

@summary 태그는 전체 설명의 짧은 버전입니다. 이는 모든 문서 항목에 추가할 수 있습니다.

## 예시

```js
/**
 * 아주 길고, 장황하고, 말이 많은, 길게 늘어선, 지루하고, 장황하며, 진부한,
 * 부적절하고, 광대한, 열정적인, 중복된, 화려한, 웅변적인, 표현력이 풍부한,
 * 수다스러우며, 자세한, 딴소리를 늘어놓는 설명입니다.
 * @summary 간결한 요약입니다.
 */
function bloviate() {}
```
