## 동의어

## 개요

@file 태그는 파일에 대한 설명을 제공합니다. 파일의 시작 부분에 있는 JSDoc 주석에서 이 태그를 사용하세요.

## 예시

파일 설명

```js
/**
 * @file 위젯의 구성 설정을 관리합니다.
 * @author Rowina Sanela
 */
```
